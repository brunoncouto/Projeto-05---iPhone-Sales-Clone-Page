# Projeto 05 - Clone página de vendas iPhone

Projeto desenvolvido durante a Formação Front-End;<br>
Tecnologias utilizadas: HTML5, CSS3 e Javascript<br>

Conceitos aplicados:<br>
  . Aprofundamento nos conceitos Javascript, como seleção de atributos HTML com querySelector; <br>
. Estruturas de repetição como For, forEach, If Else<br>
. Utilização de crase para formatação dinâmica de elementos <br>
. Aplicação de conceitos de criação de funções<br>
. Alteração de imagens na páginas, dinâmicamente<br>
. Adição e remoção de classes<br>
. Oitiva de eventos na página, como click<br>
. Aplicação de intervalos para eventos a serem realizados na página <br>
